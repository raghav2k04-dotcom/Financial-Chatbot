# Financial Chatbot

## Overview

Financial Chatbot is a Python-based intelligent assistant designed to help users manage personal finances, explore stock market opportunities, and access financial insights through an interactive chatbot interface.

The system combines financial advisory features with stock analysis and recommendation capabilities, making financial information more accessible and user-friendly.

---

## Features

### Personal Finance Assistance
- Budget planning
- Expense management guidance
- Financial literacy support
- Savings recommendations

### Stock Recommendation System
- Stock suggestions based on financial data
- Investment insights
- Market analysis support

### Stock Consulting
- Company stock analysis
- Historical stock performance review
- Investment decision support

### Financial News Integration
- Yahoo Finance article retrieval
- Financial market updates
- Investment-related news summaries

### Interactive Chatbot
- Natural language interaction
- Rule-based financial responses
- User-friendly interface

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Financial-Chatbot.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Chatbot

```bash
python main.py
```

## Run the Streamlit Application

```bash
streamlit run app.py
```
